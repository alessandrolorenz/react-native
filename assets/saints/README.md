Replace them one-by-one with the final illustrations before publishing.

## Style guide

- **Format:** PNG, square (1:1)
- **Size:** 512×512 minimum (1024×1024 preferred for high-density screens)
- **Look:** soft cartoon, pastel palette, gentle expressions, minimal background — child-friendly Catholic illustration
- **Composition:** the saint centered, head & shoulders or full bust, with breathing room around the figure (no edge-to-edge cropping)
- **Background:** simple pastel wash or transparent — avoid busy detail; the card UI adds its own framing

## Adding a new saint later

1. Drop the new PNG into this folder using the saint's `id` as the filename.
2. Append a new entry to `src/data/saints.js` referencing it via `require`.
3. The deck builder (`src/utils/deck.js`) automatically picks 8 saints at random per round, so adding more increases variety without code changes.
